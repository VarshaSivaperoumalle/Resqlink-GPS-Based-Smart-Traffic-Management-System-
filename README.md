# ResQLink – GPS-Based Smart Traffic Management System

## Overview
ResQLink is a GPS-based intelligent traffic management system designed to prioritize emergency vehicles by creating dynamic green corridors. The system uses real-time location tracking, geofencing, and automated traffic signal control to ensure faster and more efficient ambulance movement through urban traffic.

## Features
- Real-time ambulance tracking
- Automated traffic signal control
- Dynamic green corridor creation
- Route monitoring with ETA updates
- Web-based dashboard interface
- Local authentication system
- Progressive Web App (PWA) support

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Mapping: Leaflet.js
- Storage: LocalStorage
- PWA: Service Worker and Web App Manifest
